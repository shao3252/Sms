# Sms Tamu 💖

A beautifully designed, mobile-first web application for sharing, discovering, and interacting with daily love texts. Built to feel like a native social media app, Sms Tamu runs entirely offline using local storage, making it lightning-fast and highly responsive.

## ✨ Features

### 👤 User Features
* **Authentication:** Secure registration and persistent login sessions.
* **Social Interactions:** Like, comment on, and save your favorite love texts.
* **Follow System:** Follow other users and build your audience. 
* **Categorized Feed:** Filter texts by categories like *Apologies*, *Praise*, *Seductive*, and *Other*.
* **Rich Profiles:** Upload custom profile pictures, update bios, and view followers/following counts.
* **Smart Notifications:** Clickable in-app alerts for likes, comments, and admin replies that take you directly to the content.
* **Dark/Light Mode:** Seamless theme toggling saved to your preferences.
* **WhatsApp Sharing:** Share your favorite texts directly to WhatsApp with one click.
* **Support & Appeals:** Submit bug reports directly to the admin. If your account gets restricted, you can submit an appeal from the settings menu.

### 🛡️ Admin Panel Features
* **Post Moderation:** Approve, reject, or delete user-submitted posts from a dedicated queue.
* **Category Management:** Edit the category of any post on the platform.
* **User Management:** * Block/Unblock disruptive users.
    * Grant the highly-coveted "Verified" blue tick badge.
    * Add custom follower counts to users.
    * Delete accounts.
* **Global Announcements:** Broadcast messages to all users that automatically expire after 24 hours.
* **Bug & Appeal Resolution:** Reply directly to user bug reports (users get notified) and review/resolve account block appeals.

## 🛠️ Tech Stack

This project was built from scratch without any external libraries or frameworks to ensure 100% offline functionality.

* **HTML5** (Structure & Layout)
* **CSS3** (Modern UI/UX, Flexbox/Grid, Animations, Dark Mode variables)
* **Vanilla JavaScript (ES6+)** (DOM manipulation, routing, state management)
* **LocalStorage API** (Simulated NoSQL database)
